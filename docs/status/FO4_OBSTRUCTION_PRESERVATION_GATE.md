# Conditional: FO4 obstruction preservation gate

## Gate

\[
\mathrm{PreserveObstructionClass}
:= (R > \delta_{\mathrm{sep}})
\]

## Enforcement

Only if
\[
R > \delta_{\mathrm{sep}}
\]
may one assert
\[
[T_R(z)] \neq 0 \iff [z] \neq 0.
\]

## Failure

If
\[
R \le \delta_{\mathrm{sep}},
\]
preservation is not certified.

## Status

Conditional.
