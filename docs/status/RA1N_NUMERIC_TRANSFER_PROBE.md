# RA1n Numerical Transfer Probe

Status: NUMERICAL / NON-PROOF / FRONTIER AID

Full_RA1n_status: CONDITIONAL

Purpose:

Numerically test candidate RA1n interaction symbols for the terminal alternatives:

\[
g\equiv0
\]

or

\[
F\in g^\perp.
\]

This probe does not prove full RA1n closure.

It detects whether a candidate \(\sigma_{k,\omega}\) behaves compatibly with:

1. signed-pair antisymmetry,
2. weighted FM-1 transfer,
3. projected-packet cancellation,
4. the counterexample guard.

The exact obstruction is

\[
g(\eta)
=
(\xi_{\mathrm{out}}-\eta-\xi_{k,\omega})^\beta
\sigma_{k,\omega}(\eta)
J_{k,\omega}(\eta).
\]

The theorem-level guard is

\[
F=\frac{\overline g}{\|g\|_2}.
\]

A passing numerical probe is evidence only.

It is not an admissible proof of RA1n closure.
