# AKCL Tail Domination Lemma

## Empirical Law

Let
R(L) = (tail mass) / (core mass)

Observed:
R(L) → 0 as L → ∞

## Structural Statement

∃ ε(L) → 0 such that:
∑_{tail} |f|^2 w ≤ ε(L) ∑_{core} |f|^2 w

## Consequence

∑_{all} |f|^2 w ≤ (1 + ε(L)) ∑_{core} |f|^2 w

## Combined with Localization

E[f] ≥ c0 ∑_{core} |f|^2 w

⇒

E[f] ≥ c0 / (1 + ε(L)) ||f||^2

## Limit

ε(L) → 0 ⇒ uniform coercivity constant c0

## Closure

AKCL holds if tail domination is universal over admissible class
