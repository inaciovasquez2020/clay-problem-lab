# Explicit Decay Parameter

## Input

Assume strip width
\sigma(B) = 1.

Assume contour shift uses any
0 < \eta < 1.

## Decay Parameter

Define
\delta(B,\eta) = \tfrac12 + \eta.

Then
|f(e^u)| \le C(B,\eta)e^{-\delta(B,\eta)u}.

Equivalently,
|f(x)| \le C(B,\eta)x^{-\delta(B,\eta)}.

## Canonical Choice

Set
\eta = \tfrac12.

Then
\delta(B) = 1.

## Consequence

\int_L^\infty |f(x)|^2 e^{-x} dx
\le
C(B)e^{-L}L^{-2}.

## Use

This supplies an explicit rate for
\varepsilon_B(L).
