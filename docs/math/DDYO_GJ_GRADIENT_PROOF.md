# DDYO G_j Gradient Proof

## Target theorem

Prove
\[
\|\nabla(x_\ell G_j)\|_{L^\infty}\le C\,2^{-j}.
\]

## Allowed input

Use either:

1. an explicit theorem-level definition of \(G_j\); or
2. an explicit dyadic kernel representation
   \[
   G_j(x)=2^{2j}\Gamma(2^j x)
   \]
   with sufficient bounds on \(\Gamma\).
