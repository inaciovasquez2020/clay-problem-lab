# DDYO Claim 5 Final Dichotomy

## Frontier object
Let \(\mathsf T^{(1)}_{j,k}\) denote the post-symmetrized first-order leakage term.

## Required dichotomy
For all admissible near-diagonal pairs \(|j-k|\le C\), establish one of the following:

### Branch A: exact cancellation
\[
\mathsf T^{(1)}_{j,k}=0.
\]

### Branch B: lower-order routing
\[
|\mathsf T^{(1)}_{j,k}|
\le
C_\varepsilon\bigl(\Phi[u]+\lambda R^{\mathrm{cont}}_{\mathrm{DDYO}}[u]\bigr)_{j,k}.
\]

## Weakest sufficient consequence
Either branch implies that the remaining paired remainder satisfies
\[
|\mathsf R_{j,k}|
\le
C\,2^{-j}2^{-k}\,
\|\nabla S_k\|_{L^\infty}\,
\|\omega_k\|_{L^1},
\qquad |j-k|\le C,
\]
up to summable lower-order terms.

## Status
Conditional frontier.
This is the minimal remaining object for hybrid DDYO Claim 5 after first-order post-symmetrization.
