# DDYO Remainder Routing Lemma

## Candidate routing statement

Let \(\mathsf R^{\mathrm{lo}}_{j,k}\) denote all lower-order paired commutator remainders.
The weakest sufficient routing statement is
\[
\sum_{|j-k|\le C} |\mathsf R^{\mathrm{lo}}_{j,k}|
\le
C_\varepsilon\bigl(\Phi[u]+\lambda R^{\mathrm{cont}}_{\mathrm{DDYO}}[u]\bigr).
\]

## Consequence

If the terminal paired remainder satisfies
\[
|\mathsf R_{j,k}|
\le
C\,2^{-j}2^{-k}\,
\|\nabla S_k\|_{L^\infty}\,
\|\omega_k\|_{L^1},
\]
and all lower-order terms route into
\[
\Phi[u]+\lambda R^{\mathrm{cont}}_{\mathrm{DDYO}}[u],
\]
then Claim 5 closes.

## Status

This is a candidate routing lemma, not a proved claim.
