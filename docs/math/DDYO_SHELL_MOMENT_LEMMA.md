# DDYO Shell Moment Lemma

## Candidate shell moment facts

For each nonzero dyadic shell \(k\),
\[
\int_{\mathbb R^3} S_k(x)\,dx = 0,
\qquad
\int_{\mathbb R^3} \omega_k(x)\,dx = 0.
\]

## Weakest sufficient usage

These vanishing-moment identities are admissible inputs for the paired remainder gain:
\[
|\mathsf R_{j,k}|
\le
C\,2^{-j}2^{-k}\,
\|\nabla S_k\|_{L^\infty}\,
\|\omega_k\|_{L^1}.
\]

## Status

This is a candidate auxiliary lemma, not a proved claim.
