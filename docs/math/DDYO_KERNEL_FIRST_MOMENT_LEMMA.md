# DDYO Kernel First Moment Lemma

## Candidate kernel first-moment bound

For the smooth positive dyadic projector \(\widetilde P_j\) with kernel \(K_j\),
\[
\int |K_j(z)|\,|z|\,dz \lesssim 2^{-j}.
\]

## Weakest sufficient usage

Combined with shell localization, this provides the first effective dyadic gain in the paired commutator remainder analysis.

## Status

This is a candidate auxiliary lemma, not a proved claim.
