# DDYO RA1n Single Remaining Theorem

## Canonical statement

\[
\forall \alpha\in\mathbb N^3,\qquad
\sup_{k\ge 0}\sup_{|\xi|\sim 2^k}
2^{k(1+|\alpha|)}
\left|
\partial_\xi^\alpha\!\bigl(\widehat G_k(\xi)-P_k\widehat G_k(\xi)\bigr)
\right|
<\infty.
\]

## Canonical remainder

\[
r_k(\xi)=\widehat G_k(\xi)-P_k\widehat G_k(\xi).
\]

## Consequence

\[
\text{single remaining theorem}
\Longrightarrow
\text{unconditional RA1n closure}.
\]
