# AKCL Constant Sheet (Empirical)

## Observed Scaling

Let
R(L) = E[f] / ||f||^2

Empirical values:

L = 2 → R ≈ 10.43  
L = 3 → R ≈ 5.57  
L = 4 → R ≈ 2.97  
L = 5 → R ≈ 1.64  

## Invariant

R(L) ↓ as L ↑

## Weak Coercivity Form

∃ c(L) such that:
E[f] ≥ c(L) ||f||^2

with
c(L) ≈ O(L^{-1})

## Required Upgrade

Uniform coercivity:
∃ c0 > 0 independent of L
