# DR33D Step 32 — Normalization and the Exact Size-Floor Obstruction

## Status
OPEN

Normalize by
\[
\widetilde E_j:=\frac{E_j}{S^{1/(1+\theta)}}.
\]

Then
\[
\sum_j \widetilde E_j^{\,1+\theta}
\]
is normalized to 1.

The normalized amplitude is
\[
\widetilde A:=\sum_j 2^{-j}\widetilde E_j^{1/2}.
\]

The obstruction compares
\[
A^2
\]
against
\[
S^{1/(1+\theta)}.
\]

A theorem-grade closure requires
\[
S\ge m>0.
\]

The one-shell counter-scaling model is
\[
E_j=a\,\mathbf 1_{\{j=j_\ast\}}.
\]

LOCKED NORMALIZATION — obstruction is exactly the vanishing-size regime \(S\to0\); linear upgrade requires a positive size-floor invariant.
