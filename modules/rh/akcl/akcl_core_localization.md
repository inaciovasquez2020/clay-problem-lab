# AKCL Core Localization

## Observation

Tail mass T(L) → 0 as L → ∞

## Single Structural Upgrade

Restrict to core region:

Core(L) = { x ≤ θ L }, 0 < θ < 1

## Localized Inequality

E[f] ≥ c0 ∑_{x∈Core(L)} |f(x)|^2 w(x)

## Reduction

Global coercivity ⇔ transfer of mass from tail to core

## Required Lemma

For all admissible f:

∑_{x∈Tail(L)} |f(x)|^2 w(x) ≤ ε(L) ∑_{x∈Core(L)} |f(x)|^2 w(x)

with ε(L) → 0

## Closure

Localized coercivity + tail domination ⇒ uniform coercivity
