# DDYO First-Order Leakage Split

## Sole unresolved object

For \(|j-k|\le C\), let
\[
\mathsf T^{(1)}_{j,k}
\]
denote the post-symmetrized first-order Taylor leakage in the paired hybrid route.

## Two admissible closure branches

### Branch A: exact vanishing
Prove
\[
\mathsf T^{(1)}_{j,k}=0.
\]

### Branch B: lower-order routing
Prove
\[
|\mathsf T^{(1)}_{j,k}|
\le
C_\varepsilon\bigl(\Phi[u]+\lambda R^{\mathrm{cont}}_{\mathrm{DDYO}}[u]\bigr)_{j,k}.
\]

## Weakest sufficient outcome

Either Branch A or Branch B is sufficient to conclude that the remaining paired remainder satisfies
\[
|\mathsf R_{j,k}|
\le
C\,2^{-j}2^{-k}\,
\|\nabla S_k\|_{L^\infty}\,
\|\omega_k\|_{L^1},
\qquad |j-k|\le C,
\]
up to summable lower-order terms.

## Consequence

This reduces the hybrid DDYO Claim 5 frontier to one final dichotomy:
exact vanishing versus admissible routing of the post-symmetrized first-order leakage.
