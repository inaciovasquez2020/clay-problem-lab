# RA1n Terminal Grammar Atom Span Lemma

Status: OPEN.

## Lemma

Let \(\mathfrak G_{\mathrm{atom}}\) be the set of primitive atoms in the actual RA1n terminal grammar.

\[
\forall \gamma\in\mathfrak G_{\mathrm{atom}},\quad
\gamma\in W_{\mathrm{term}}.
\]

Equivalently,

\[
\forall \gamma\in\mathfrak G_{\mathrm{atom}},\quad
(I-\Pi_W)\gamma=0.
\]

Equivalently,

\[
\forall \gamma\in\mathfrak G_{\mathrm{atom}},\quad
\mathcal R_{\mathrm{term}}(\gamma)=0.
\]

## Rebuild target

If every primitive terminal grammar atom lies in \(W_{\mathrm{term}}\), then the remaining work is only to prove that every admissible terminal generator is obtained from these atoms by operations preserving \(W_{\mathrm{term}}\).
