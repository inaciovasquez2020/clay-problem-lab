# DR33D Step 16 — Minimal Invariant Axiom Schema

## Objective
State the weakest abstract invariant sufficient to close the DR33D bridge.

## Data
Let \(E=(E_j)_j\) be a nonnegative dyadic shell profile.

## Axiom schema
Assume there exists a functional \(\mathfrak I(E)\ge 0\) and constants \(C_1,C_2>0\), \(\theta\in(0,1)\), such that for every admissible shell profile \(E\),
\[
\left(\sum_j 2^{-j}E_j^{1/2}\right)^2
\le
C_1\,\mathfrak I(E),
\]
and
\[
\mathfrak I(E)
\le
C_2\sum_j E_j^{1+\theta}.
\]

## Immediate consequence
Then
\[
\left(\sum_j 2^{-j}E_j^{1/2}\right)^2
\le
C_1C_2\sum_j E_j^{1+\theta}.
\]

## DR33D closure consequence
The Step 6 subquartic bridge holds.

## Logical role
This schema isolates the exact missing ingredient independently of any specific realization of \(\mathfrak I\).
