# Proof Status

| Field | Value |
|---|---|
| Claim surface | Clay Problem Lab |
| Status | Experimental and artifact-level framework. |
| Scope | Experimental framework for computational exploration, artifact generation, dashboards, and reproducible problem-lab workflows. |
| Non-claim | No theorem-level completion claim is made by this repository unless a theorem-level document states it explicitly. |

## Status discipline
- `Proved` = theorem-level claim closed inside declared scope.
- `Conditional` = depends on a named external hypothesis or unresolved lemma.
- `Executable` = build/test/verification surface closed without implying full theorem-level closure.
- `Experimental` = computational or exploratory artifact only.
