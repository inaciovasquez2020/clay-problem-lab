# DDYO Post-Symmetrized First-Order Leakage

## Sole next terminal target

For \(|j-k|\le C\), let
\[
\mathsf T^{(1)}_{j,k}
\]
denote the first-order Taylor contribution to the paired commutator remainder after kernel symmetrization in the hybrid route.

The sole next target is to prove either
\[
\mathsf T^{(1)}_{j,k}=0,
\]
or the weaker sufficient routing bound
\[
|\mathsf T^{(1)}_{j,k}|
\le
C_\varepsilon\bigl(\Phi[u]+\lambda R^{\mathrm{cont}}_{\mathrm{DDYO}}[u]\bigr)_{j,k}.
\]

## Weakest sufficient consequence

If the post-symmetrized first-order leakage vanishes or routes to lower order, then the remaining paired remainder satisfies
\[
|\mathsf R_{j,k}|
\le
C\,2^{-j}2^{-k}\,
\|\nabla S_k\|_{L^\infty}\,
\|\omega_k\|_{L^1},
\qquad |j-k|\le C,
\]
up to summable lower-order terms.

## Status

This isolates the only unresolved first-order leakage object in the hybrid paired/Taylor/kernel route.
