# Terminal High-High Resonance Curvature Gain Lemma

## Status
OPEN

## Statement
There exist constants c > 0, alpha >= 0, a localized high-high support region Omega_k, and an exact closed-form DDYO interaction symbol m_k(xi,eta) for
r_k = \widehat G_k - P_k \widehat G_k
such that, for the transverse renormalized symbol
\widetilde r_k(xi,eta) := m_k(xi,eta) - m_k(xi,eta)|_{xi || eta},
one has
|\widetilde r_k(xi,eta)| >= c 2^{-alpha k} |xi \wedge eta|^2
for all (xi,eta) in Omega_k.
